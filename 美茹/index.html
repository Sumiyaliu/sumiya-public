📏 排查门槛 / Threshold

多素材使用同主商品在ADQ链路中投放时，消耗极度分化，新计划几乎“零展现”，遭遇ADQ严格的同质化过滤。

💡 原因解读 / Diagnosis

系统为了规避流量空转及商户内卷，对ADQ同一商品的直购/CID链路有强唯一性判准（画面、落地页、商品ID相似度超过阈值即拦截）。

✅ 优化行动 / Action Plan

  • •微调物理规格：如改用“一周期装/多杯多规格”，微调商品单价；
  • •增加考核单元要素：调整落地页细节，增加优惠信息微调其主ID；
  • •差异化前3秒：对视频画面和开场镜头做显著区分。
🔴 痛点排查 2：审核屡投屡拒，被系统拦截夸大疗效、敏感词雷区 高频雷区

📏 排查门槛 / Threshold

视频或直播间话术中频繁提及某些日常词，甚至仅对传统成分作功能描述，便被判定为“非药品涉及暗示医疗或防病效果”。

💡 原因解读 / Diagnosis

广告法及平台大健康规范严禁食品/滋补品暗示具体疗效（如“排湿、去根、降糖、消炎”）。即使引用中医成分，也极易在OCR/语音审核中触发自动屏蔽。

✅ 优化行动 / Action Plan

  • •合规话术替换：将“除湿去湿气”改用“伏天调理、养水湿”；将“消炎消暑”改用“清火不腻”；
  • •利用官方白名单：素材多科普成分本身，如“据神农本草记载...”避开直白暗示；
  • •自查视频号指引：使用后台提供的「审核拒绝修改指引」快速微调。
