# 公开站点

> 这里放对外公开的 HTML 报告 / 案例 / 创意双周报等。

**访问地址**：<https://Sumiyaliu.github.io/sumiya-public/>

## 目录

- `index.html` — 门户页（站点首页）
- `creative-biweekly-report.html` — 传统滋补 & 保健食品创意双周报（2026-06-26）

## 更新方式

1. 把新的 HTML / 图片放进本目录
2. 终端执行：
   ```bash
   cd "/Users/sumiyaliu/Documents/sumiya的AI小森林/公开"
   git add .
   git commit -m "update: 简短说明"
   git push
   ```
3. 1 分钟后 GitHub Pages 自动重新部署

## 文件命名规范

新加 HTML **务必用英文 / 拼音文件名**（如 `report-2026-07.html`、`case-xxx.html`），
避免中文 URL 编码后链接长且不友好。

## 注意事项

- 不放客户名 / 投放数据 / 未脱敏内容
- HTML 引用的图片放在同目录的相对路径下
- 仓库是 Public，发内容前请过一遍敏感词
