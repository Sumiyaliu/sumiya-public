# 公开站点

> 这里放对外公开的 HTML 报告 / 案例 / 创意双周报等。

**访问地址**：<https://Sumiyaliu.github.io/sumiya-public/>

## 目录

- `传统滋补保健食品创意双周报.html` — 传统滋补 & 保健食品创意素材双周报

## 更新方式

1. 把新的 HTML / 图片放进本目录
2. 终端执行 `git add . && git commit -m "update" && git push`
3. 1 分钟后 GitHub Pages 自动重新部署

## 注意事项

- 不放客户名/投放数据/未脱敏内容
- HTML 引用的图片放在同目录的相对路径下
