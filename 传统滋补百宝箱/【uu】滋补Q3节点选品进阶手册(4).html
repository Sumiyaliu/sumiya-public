一、本期主题聚焦：「三伏滋养季」整体打法（7月中-8月中）
六大主题已在第〇部分全景列出，本手册只重点讲透 Q3 体量最大的「三伏滋养季」这一个主题。三伏在 Q3 是一个完整的趋势季，这里不再拆初伏/中伏/末伏——给您留足备货与素材准备时间。三伏滋养下分 清热润燥 / 祛湿养阳 / 温补理脏 三个并行方向——这不是客单价高低之分，而是三种不同的调养思路，三者之间没有绝对的客单价区隔。人群以视频号画像为准，聚焦中老年与全年龄段。颜色：清热润燥 祛湿养阳 温补理脏
1.1 三伏滋养季 ｜ 三个调养方向一图速览
读法：三伏滋养季围绕「夏季养生」做主线，下分三个并行方向（沿用清热/祛湿/温补三套调养思路）。类目参考视频号《药食同源准入清单》归类，被清单明确收录的成分归入对应类目（绝大多数草本归「药食同源食品」），清单未单列对应关系的品类（如燕窝、阿胶）单独摘出。建议在整个 7月中-8月中 不要只跑某一个方向，而是把多个方向结合起来铺货，无需按伏天细拆。

三伏滋养下分三个并行方向——清热润燥 祛湿养阳 温补理脏，列出每个方向下有哪些品可以推。这三个方向不是客单价高低之分，建议多个方向结合着跑；整季汇总、不拆初伏中伏末伏；默认目标人群即视频号中老年 / 全年龄段。「类目 / 成分」一列参考视频号《药食同源准入清单》归类：清单明确收录的成分归入对应类目（独立类目如石斛 / 人参·西洋参 / 灵芝·孢子粉 / 阿胶 / 燕窝等，其余草本统一归「药食同源食品」并把具体成分如茯苓标在后面）；清单里没有给出对应关系的品类（如燕窝、阿胶单算一类，五指毛桃等未收录成分），就单独摘出来标注，便于对照投放类目。每个品同时标注两个数据源：抖音 三伏选品方向 + 视频号 友望实测爆品，覆盖药食同源 / 枸杞 / 燕窝 / 参类 / 阿胶 / 石斛 / 三七等多品类。
