## 二、模块架构（6模块）

### 模块1：核心政策 — 创意唯一性过滤竞争

**认知层（lb-change）**：
- 核心变化：同一客户下，按「创意唯一性」内容参与竞争，其余过滤
- 核心公式：`创意唯一性ID = 广告维度多样性 × 素材指纹差异化`
- 关键结论：创意唯一性数量越多样 = 竞价机会越多 = 流量越多

**行动层（lb-action）**：
- P0：接入妙思托管 + 多OG并行
- !：视频号广告请在每日10:00前创建

**布局**：全宽模块，左右两栏（flex:1.2 + flex:0.8）

**mod-h 渐变**：`linear-gradient(135deg,#dc2626,#ef4444)`

### 模块2：两列 — 广告维度多样性 / 素材指纹差异化

**左列 — 广告维度多样性**：
- 认知层：不同的优化目标、营销对象、目的会生成不同的广告维度
- 行动清单：
  - P0 多OG并行 — 下单/下单-ROI/24h下单等多优化目标
  - P0 直播间增加商品表达 — 单品直播间挂接小店商品，找运营经理开白
  - P1 专场放量 — 选择「专场放量」营销目的
  - P1 测试自动出价 — 部分广告测自动出价
- 商家Todo：当前广告是否只用了单一OG？→ 增设多OG并行 → 检查投放主体是否可拆分 → 尝试专场放量
- mod-h 渐变：`linear-gradient(135deg,#2563eb,#3b82f6)`

**右列 — 素材指纹差异化**：
- 认知层：不同的素材内容形态（规格、创意）会生成不同的素材指纹
- 行动清单：
  - P0 接入妙思托管 — AIGC自动衍生差异化创意指纹
  - P1 丰富素材规格 — 竖版转横版
  - P1 开启衍生MAX — 为AIGC衍生预留空间
- 商家Todo：手动制作大量素材，无需重复微调
- mod-h 渐变：`linear-gradient(135deg,#16a34a,#22c55e)`

### 模块3：执行路线 — 从「堆量」转向「提质」

时间线组件（4步）：
1. Step1 多OG+多主体并行（红点，P0 立即执行）
2. Step2 接入妙思托管（橙点，P0 立即执行）
3. Step3 丰富素材形态（绿点，P1 逐步优化）
4. Step4 拓展广告维度（绿点，P1 逐步优化）

**mod-h 渐变**：`linear-gradient(135deg,#f59e0b,#fbbf24)`，color:#78350f

#### 4b. 潜客优投

- 标签：lb-action + NEW标签（红底白字）
- 核心说明：帮助小店商家高效触达「高购买意向潜客」
- KPI卡片：提价系数范围 1.1~2.0 / 适用投放模式 AIM+ & 竞价 / 功能开通方式 需加白
- 出价公式：ROI类 调价后出价=原始出价/调价系数；非ROI类 调价后出价=原始出价×调价系数
- 商家Todo：联系运营经理申请加白 → 新建/编辑项目 → 出价与预算模块开启探索策略 → 选择潜客优投 → 设置提价系数（建议1.2~1.5）

- 二维码：潜客优投指南.png → 潜客优投对客说明

**内容结构**：

- 传统滋补 CPM 趋势折线图（近7天，Plotly 交互）
- 数据口径说明

**传统滋补分版位核心数据**：

| 版位（MP二级流量） | CPM（元） | CTR | CVR | 下单ROI |
|:---|:---|:---|:---|:---|
| 整体 | 80~100 | 2%~3% | 6%~8% | 2.0~2.5 |
| 微信搜一搜 | 150+ | 7%+ | <2% | 1.0~1.5 |
| 微信小程序 | 80~100 | 3%~5% | 6%~8% | 2.0~2.5 |
| 微信视频号 | 80~100 | 2%~3% | 6%~8% | 1.5~2.0 |
| 微信公众号 | 40~60 | 1%~2% | 2%~4% | 2.0~2.5 |
| 微信朋友圈 | 40~60 | 1%~2% | 4%~6% | 4.0+ |

**关键洞察**：
- 微信朋友圈：ROI最高（4.0+），CPM低（40~60），是传统滋补闭环高价值版位
- 微信小程序：CTR优秀（3%~5%），ROI稳定（2.0~2.5），优先分配预算
- 微信视频号：ROI相对较低（1.5~2.0），需结合直播实时画面能力提升转化
- 微信搜一搜：高意图流量但CVR低（<2%），建议配合品牌词定向使用
