右 · 冲量宝 · 新营销快速起量NEW

为智投项目单独配置冲量预算，系统在 6 小时内快速消耗预算。

适用场景
  • 1. 智投项目新投放初期，无法快速获取曝光量和转化量
  • 2. 线索量级缺口较大，需在指定时段内快速补量
  • 3. 大促营销场景下，加速预算消耗
使用方法

新建/编辑营销时开启冲量宝，设定冲量预算 + 成本上浮比例。

冲量预算
200~100000
成本上浮比例
20%~300%
投放建议：初期可阶梯化设置预算测试；数据量充分可直接开启；数据量少建议积累 3~5 个转化后再开启。
商家Todo：新建/编辑智投项目 → 开启冲量宝 → 设置冲量预算（建议为浅层出价的 10 倍） → 设置可接受成本上限（建议 ≥ 20%） → 监控 6 小时内消耗与转化情况
模块3 · 创意重复度与唯一性说明
唯一性规则「4+m」

营销目的 + 推广产品 + 营销载体 + 优化目标（4 个固定参数），在版位 / 定向 / 出价（m 个变量）都一致时，仅能创建 1 条广告。

创意重复（雷同）判定标准：系统依据视频画面相似、片段混剪重合、图文语义相似等进行底层去重过滤，雷同素材在竞价池外自动合并。
🟢 DOs 推荐操作
  • • 多 OG 并行（下单 / 下单-ROI / 下单-深辅）
  • • 接入妙思托管自动衍生差异指纹
  • • 改变素材规格打破物理指纹
🔴 DON'Ts 规避操作
  • • 多账户铺同一视频触发跨主体排重
  • • 单广告内塞满微调画面创意
  • • 仅靠改亮度 / 加 Logo 去重
商家Todo：多 OG 并行（下单 / 下单-ROI / 下单-深辅） → 7 天内 TID ≥ 150、重复度 < 15% → 补充差异化素材、开启冲量宝加速起量
模块4 · 行动路线图
第一阶段 · 基建搭建0-7天

确认投放链路 → 申请全库智选白名单 → 补充创意素材 TID ≥ 150 → 开启商品智投

第二阶段 · 加速起量7-14天

开启冲量宝快速拿量 → 多 OG 并行测试 → 监控成本 → 阶梯化调整冲量预算

第三阶段 · 持续优化持续

筛选最优优化目标组合 → 核心素材持续迭代 → 冲量宝与常规投放协同 → ROI 稳定提升
