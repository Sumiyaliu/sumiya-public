冲量宝 · 新营销快速起量

产品能力：为智投项目单独配置冲量预算，系统在 6 小时内快速消耗预算。

适用场景

  • 智投项目新投放初期，无法快速获取曝光量和转化量
  • 线索量级缺口较大，需在指定时段内快速补量
  • 大促营销场景下，加速预算消耗

使用方法：新建/编辑营销时开启冲量宝，设定冲量预算 + 成本上浮比例。

投放建议：初期可阶梯化设置预算测试；数据量充分可直接开启；数据量少建议积累3-5个转化后再开启。
商家Todo：新建/编辑智投项目 → 开启冲量宝 → 设置冲量预算（建议为浅层出价的10倍）→ 设置可接受成本上限（建议≥20%）→ 监控6小时内消耗与转化情况
模块3 · 创意重复度与唯一性说明
唯一性规则「4+m」：营销目的 + 推广产品 + 营销载体 + 优化目标（4）在 版位 / 定向 / 出价（m）都一致时，仅能创建 1 条广告

创意重复（雷同）判定标准

在广告维度（4）与版位/定向/出价（m）高度一致时，系统判定为雷同创意并予以过滤，不计入有效竞量。

DOs ✓

  • 多OG并行（下单/下单-ROI/下单-深辅）
  • 补充差异化素材
  • 开启冲量宝加速起量

DON'Ts ✗

  • 同一素材反复微调充数
  • 单一OG堆叠大量雷同广告
  • 忽视重复度直接放量
商家Todo：多OG并行（下单 / 下单-ROI / 下单-深辅） → 7天内 TID≥150、重复度<15% → 补充差异化素材、开启冲量宝加速起量
模块4 · 行动路线图
第一阶段 · 0-7天基建搭建

确认投放链路 → 申请全库智选白名单 → 补充创意素材 TID≥150 → 开启商品智投

开启冲量宝快速拿量 → 多OG并行测试 → 监控成本 → 阶梯化调整冲量预算

第三阶段 · 持续持续优化

筛选最优优化目标组合 → 核心素材持续迭代 → 冲量宝与常规投放协同 → ROI稳定提升
