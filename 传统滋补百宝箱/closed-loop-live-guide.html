认知层 · lb-change

核心变化：同一客户下，按「创意唯一性」内容参与竞争，其余过滤。

模块5 · 创意重复度与唯一性说明
唯一性规则「4+m」：营销目的 + 推广产品 + 营销载体 + 优化目标（4）在 版位 / 定向 / 出价（m）都一致时，仅能创建 1 条广告

创意重复（雷同）判定标准

在广告维度（4）与版位/定向/出价（m）高度一致时，系统判定为雷同创意并予以过滤，不计入有效竞量。

DOs ✓

  • 多OG并行，分散优化目标
  • 素材规格与创意形态差异化
  • 接入妙思托管自动衍生指纹

DON'Ts ✗

  • 同一素材反复微调充数
  • 单一OG堆叠大量雷同广告
  • 重复挂接相同商品表达
商家Todo：账户深耕（1-2账户聚焦70%预算） → 7天内 TID≥150、重复度<15% → 接入妙思托管 + 多OG并行
